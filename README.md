This project presents an interactive HR Analytics Dashboard built using Power BI to help organizations analyze key workforce metrics and make data-driven HR decisions. The dashboard provides insights on employee attrition, headcount, performance, hiring trends, salary structure, and diversity.

--Project Overview

The goal of the dashboard is to understand and improve workforce performance by analyzing:

Attrition trends

Employee headcount

Hiring & recruitment metrics

Salary distribution

Performance ratings

Diversity insights (gender, experience, department)

--Key Insights

Identified departments and job roles with the highest attrition

Analyzed how factors like age, salary, education, and job role influence attrition

Visualized performance ratings and team-wise distribution

Examined gender and experience level diversity

Highlighted salary patterns across different roles and departments

 --Tools & Techniques

Power BI for data modeling & visualization

Power Query for data cleaning and transformation

DAX measures for KPI calculations

Excel / CSV datasets
